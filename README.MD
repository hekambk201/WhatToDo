## About The Project

<img src="/screenshots/logo.png" width="100" />

What to Do is a task management mobile app that has three main features: Calendar, Tasks, Notes

### Built With

- [Flutter](https://flutter.dev/)
- [Firebase](https://firebase.google.com/)
